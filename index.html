Currencify

      How to use

      1. Input Amount: Enter the amount you want to convert in the "Amount" input field.
      2. Select Base Currency: Start typing or click on the "From" input field to choose the base currency. You can search for the currency by name or code.
      3. Select Target Currency: Similarly, choose the target currency in the "To" input field.
      4. Conversion: Click on the "Convert" button. The converted amount and exchange rate will be displayed.
      5. Swap Currencies (Optional): If needed, you can click on the "Swap" button to switch the base and target currencies.
      6. Reset (Optional): To reset the inputs and results, you can click on the "Reset" button.

      Click the "OK" button to close this dialog.
