Currencify

How to use

  1. Input Amount: Enter the amount you want to convert in the "Amount" field.
  2. Select Base Currency: Type or click "From" to choose the base currency by name or code.
  3. Select Target Currency: Choose the target currency in the "To" field.
  4. Conversion: Click "Convert" to see the converted amount and exchange rate.
  5. Swap: Click the swap arrow to switch the base and target currencies.
  6. Reset: Click "Reset" to clear all fields and results.
