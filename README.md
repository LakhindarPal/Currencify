# Currencify - Currency Converter

Currencify is an highly performant web application for instantaneous currency conversion with a modern UI.

## Features

- **Desktop Sidebar:** Automatically displays a persistent "How to use" guide alongside the converter on larger screens.
- **Modern UI:** A clean, easy-to-use interface with smooth animations and polished visual elements.
- **Smart Searching:** Quickly search and filter through 40+ supported currencies as you type.
- **Conversion History:** Automatically remembers your last 20 currency conversions so you can easily reference them.

## Live Demo

Explore Currencify live on [Glitch](https://currencify.glitch.me).

## Built With

- **HTML, CSS, and JavaScript:** Built using modern web standards without heavy frameworks.
- **Abstract API:** Powered by [Abstract API](https://app.abstractapi.com/users/signup) for reliable, live currency exchange data.

## Getting Started

### Prerequisites

- **Abstract Currency API Key:** Create a free account and get an API key from [AbstractApi.com](https://app.abstractapi.com/users/signup).
- **Web Browser:** Any modern web browser (e.g., Chrome, Safari, Firefox).

### Installation

1. **Clone the project files:**

   ```bash
   git clone https://github.com/LakhindarPal/Currencify.git
   ```

2. **Open the project folder:**
   ```bash
   cd Currencify
   ```
3. **Add your API Key:**
   Open `main.js` in a text editor and paste your Abstract API token into the `API_KEY` variable at the top of the file.

4. **Run the app:**
   Start a local web server (for example, by running `npx serve .` or `python -m http.server` in your terminal) to open the app in your browser!
